Welcome to Weeble, an anime guessing game by illdie and Birthday Boy!

Rules and Important Info

The way Weeble works is not entirely comparable to Wordle and other Wordle variants, so please take note of the following:

  • While word-based categories (sources, format, tags, studios, etc.) can be guessed exactly, number-based categories (year, episode count, etc.) can only be guessed down to a certain threshold. Being able to guess the exact number value of an anime frequently makes the game a bit too easy.
  • Each incorrect guess will reveal one more tag of the correct anime, in order of relevancy. Don't worry–spoiler tags will NOT be shown.
